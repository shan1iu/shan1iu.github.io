1. 正则匹配

题目:

  • format("=g哈哈哈="); 输出 \<g\>哈哈哈\</g\>

  • format("=g哈哈哈=, =g咳咳咳="); 输出 \<g\>哈哈哈\</g\>\<g\>咳咳咳\</g\>

    1
2
3
4
5

    format = (input) => {
return input.replace(/=g([^=]*)=/g,"<g>$1</g>")
}
format("=g哈哈哈=");  // "<g>哈哈哈</g>"
format("=g哈哈哈=, =g咳咳咳=");  // "<g>哈哈哈</g>, <g>咳咳咳</g>"

2. 去除数组中连续的item

题目

  • merge([3,2,2,4,5,5,6,2,1]); 输出 [3, 2, 4, 5, 6, 2, 1]

    1
2
3
4
5
6

    const merge = arr => {
return Array.isArray(arr) ? arr.filter((item, index) => {
return item !== arr[index + 1]
}) : []
} // 可去掉return
merge([3,2,2,4,5,5,6,2,1]);  // [3, 2, 4, 5, 6, 2, 1]
