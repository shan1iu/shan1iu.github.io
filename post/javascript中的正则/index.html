正则表达式相关方法

1. exec

1

regexObj.exec(str)

一个在字符串中执行查找匹配的 RegExp 方法，它返回一个数组, 未匹配到则返回 null.

2. test

1

regexObj.test(str)

一个在字符串中测试是否匹配的 RegExp 方法，它返回 true 或 false.

3. match

1

str.match(regexp)

一个在字符串中执行查找匹配的 String 方法，它返回一个数组或者在未匹配到时返回 null.

4. search

1

str.search(regexp)

一个在字符串中测试匹配的 String 方法，它返回匹配到的位置索引，或者在失败时返回-1

5. replace

1

str.replace(regexp|substr, newSubStr|function)

一个在字符串中测试匹配的 String 方法，它返回匹配到的位置索引，或者在失败时返回-1

当使用字符串作为第一个参数的时候, 只有第一个匹配项会被替换.

6. split

1

str.split([separator[, limit]])

一个使用正则表达式或者一个固定字符串分隔一个字符串，并将分隔后的子字符串存储到数组中的 String 方. limit 是一个整数，限定返回的分割片段数量.
